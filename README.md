# tower_defense

Jogo final da disciplina de Jogos (5º semestre, Insper).

Projeto Unity. A raiz do repositório é a raiz do projeto Unity — abra esta pasta no Unity Hub.

## Como rodar

1. Instale o Unity Hub e abra a versão indicada em `ProjectSettings/ProjectVersion.txt`.
2. No Unity Hub: **Add → Open**, aponte para a raiz deste repositório.
3. Abra a cena inicial em `Assets/Scenes/`.

## Estrutura

- `Assets/` — código, cenas e assets do jogo (versionado)
- `Packages/` — manifesto de pacotes Unity (versionado)
- `ProjectSettings/` — configurações do projeto (versionado)
- `Library/`, `Temp/`, `Logs/`, `UserSettings/` — geradas pelo Unity (ignorados)
